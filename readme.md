# 🏥 Healthcare Problem Statements in Python

## 📖 Overview

This repository contains two healthcare-based Python applications developed as part of a programming assignment. The projects demonstrate the implementation of real-world business logic using Python's `if-elif-else` statements while following software development practices using Git and GitHub.

---

## 📂 Projects Included

### 1. Healthcare Patient Risk Assessment System

A Python application that evaluates a patient's health condition based on:

- Patient Age
- Body Temperature
- Heart Rate
- Oxygen Saturation (SpO₂)

The system classifies patients into:

- Critical Risk
- High Risk
- Medium Risk
- Low Risk

Additional Feature:
- Senior Citizen Priority Consultation

---

### 2. Hospital Health Insurance Eligibility System

A Python application that determines a patient's insurance eligibility based on:

- Insurance Status
- Policy Status
- Treatment Cost
- Patient Age

The system decides whether:

- Cashless Treatment is Approved
- Insurance Approval is Pending
- Insurance is Rejected
- Self Payment is Required

Additional Feature:
- Senior Citizen Benefit with Hospital Service Discount
